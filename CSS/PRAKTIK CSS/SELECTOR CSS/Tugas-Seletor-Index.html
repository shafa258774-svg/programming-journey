Mengenal Levi Ackerman

Levi adalah kapten Pasukan Khusus Survey Corps, sebuah unit militer elit yang bertugas melawan para Titan, makhluk raksasa yang mengancam umat manusia.

Ia dikenal luas sebagai “Humanity’s Strongest Soldier” atau “Prajurit Manusia Terkuat”.
Ia memiliki sifat dingin, serius, dan perfeksionis, termasuk sangat memperhatikan kebersihan.
